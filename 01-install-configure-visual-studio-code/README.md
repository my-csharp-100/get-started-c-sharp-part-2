**.NET SDK 和 .NET 运行时**

* .NET 运行时是运行 C# 应用程序所需的代码库。 还可能会看到称为公共语言运行时 (CLR) 的 .NET 运行时。 编写 C# 代码不需要 .NET 运行时，但实际运行 C# 应用程序需要它。
* Visual Studio Code 使用 .NET SDK 和 C# 扩展为编写、运行和调试 C# 应用程序提供开发环境。
